Intro to Git:

  • What is git? Git is a distributed version control system that lets you track changes in a set of files
  • What is github? GitHub is a hosting service for software development and version control that uses Git
  • What does git do? Tracking code changes, Tracking who made changes, Coding collaboration
  • Our Project: Check out our GitHub and see if you can figure out some of our "2 Truths and a Lie"!!
Some terminal commands
  • git cd [directory name]: to change the directory
  • git checkout -b “username”: to create new branch
  • git commit -m “message”: to save all the changes on the current branch
  • git push: a command to send the changes to the main branch and save it to the repository

Intro to Python

  • What is Python? A high level programming language that’s designed to be easy to read and easy to implement.
  • Importance: Some of the very famous application created based on python ex.instagram, and it is widely used in the tech field.

Here is a simple guessing game you can use to learn Python:

Test this out on your own (maybe on an online Python IDE) and see the magic of coding!!

                import random

                # Generate a random number between 1 and 100
                answer = random.randint(1, 100)
                
                # Ask the user to guess the number
                guess = int(input("Guess a number between 1 and 100: "))
                
                # Keep track of the number of guesses
                guesses = 1
                
                # Loop until the user guesses the correct number
                while guess != answer:
                    if guess < answer:
                        print("Too low! Try again.")
                    else:
                        print("Too high! Try again.")
                    guess = int(input("Guess a number between 1 and 100: "))
                    guesses += 1
                
                # Print a message when the user guesses the correct number
                print("Congratulations! You guessed the correct number in", guesses, "guesses.")

Intro to Web Dev: HTML & CSS

  • What is HTML and CSS? HTML (the Hypertext Markup Language) and CSS (Cascading Style Sheets) are two of the core technologies for building Web pages. HTML provides the structure of the page, CSS the (visual and aural) layout, for a variety of devices. Along with graphics and scripting, HTML and CSS are the basis of building Web pages and Web Applications. (source: W3C)
  • Why should you learn it? You can use HTML to create your own websites for class projects, resumes etc.
